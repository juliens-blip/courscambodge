# Study Content Curator

## Overview

Use this skill to maintain content quality inside a study product. It keeps raw revision sheets intact while expanding the same material into flashcards and quiz questions that cover the full chapter scope.

## Workflow

1. Gather the current source set:
   - raw text notes
   - PDFs
   - deployable PDF copies when the app should link to them
   - existing flashcards
   - existing quiz questions
2. Preserve the fiche text in a raw block instead of rewriting it into a different voice.
3. Build coverage across:
   - definitions
   - examples
   - theories
   - risks and limits
   - exam questions
   - local Cambodia examples when present
4. Create flashcards that stay short and quiz items that explain the correction.
5. If a source is missing, document the gap instead of pretending it exists.

## Output rules

- Keep chapter ids stable once the app uses them.
- Keep quiz answers objectively checkable.
- Spread cards and quiz items across all loaded chapters.
- Avoid vague cards such as "What is chapter 4 about?".
- Prefer answer text that helps with exam wording.
