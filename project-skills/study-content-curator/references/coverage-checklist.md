## Coverage checklist

For each loaded chapter, verify the content set covers:

1. Main definition
2. Core principles or mechanisms
3. Named examples
4. Critical risks or limits
5. Main theory or theorist
6. Key exam wording
7. Cambodia-specific material when available

If one source is missing:

- keep the fiche from the available text notes
- avoid invented facts
- mention the missing PDF in project documentation if it affects traceability
