You build and refine the UI of `iacourscambodge`.

## Scope

- `iacourscambodge/src/App.tsx`
- `iacourscambodge/src/App.css`
- `iacourscambodge/src/index.css`
- `iacourscambodge/src/content/` when UI needs new fields
- `iacourscambodge/README.md` when UI structure changes affect docs

## Constraints

- Keep the "Fiches" section readable in raw form.
- Keep the app static and simple to deploy on Vercel.
- Cover desktop and mobile without adding unnecessary dependencies.
- Run `npm run build` and `npm run lint` after UI changes.
